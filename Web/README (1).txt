Programa web

O código do web no github só esta com o front end

abrir os arquivos html irá mostrar o visual de cada página

para ver as funcionalidades acesse o link https://apex.oracle.com/pls/apex/pimiv/r/hotel12/home?session=115785875083723

ou no link https://apex.oracle.com/pls/apex/pimiv/r/hotel12/login?session=115785875083723 pra criar a conta

A primeira página a ser exibida é a pagina de login, com os botões acessar e criar conta(caso nao esteja cadastrado)
se ja estiver cadastrado basta inserir as informações da conta nos campos e clickar em acessar

Depois é levado para a tela com uma tabela mosrtando os quartos para reservar
clickando em Reservar na linha de um quarto o site redireciona para a tela de reserva daquele quarto

Ao inserir a data de chegada e a data de saida o banco ira ver se o quarto esta disponivel e caso esteja
clickando em confirmar a reserva estara feita

Na parte superior possui 3 botoes (inicio, informações da conta e reservas)

o botão inicio leva para os quartos, o informações da conta para o local que é possivel ver e alterar
as informações da conta

para mudar algo em sua conta é preciso clickar no lapis ao lado das informações dela, depois disso
é preciso apenas alterar o campo desejado e clickar em aplicar alterações

na aba reserva mostra todas as reservas do cliente, e as que ainda estao pendentes podem ser canceladas clickando no campo cancelar ao lado da reserva
